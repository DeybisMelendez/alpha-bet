# Alpha Bet

Alpha Bet es una plataforma de pronosticos de partidos de futbol de uso personal para usarse como guía de apuestas deportivas de futbol.

La plataforma utiliza su propio sistema Elo para todos los equipos disponibles.

Se utiliza la api https://www.football-data.org/ para obtener información de partidos, competiciones, equipos y resultados, cuando un partido finaliza se debe actualizar el Elo de los equipos.

Cada partido nuevo obtenido se debe calcular su pronóstico.
