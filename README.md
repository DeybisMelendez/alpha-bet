# alpha-bet

Alpha Bet es una plataforma personal para realizar pronósticos de resultados de partidos de futbol utilizando un sistema propio de Elo combinado con algunas estrategias.

Desarrollado en Django 6.0 y utilizando la api de api-football.com (api-sports.io v3)
como única fuente para obtener información de partidos, competiciones, equipos y resultados.
